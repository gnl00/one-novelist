# 副本一时间线

## 场景

混乱世界X

## 副本设定
- **世界调性**：现代末世+超自然危机。城市高楼林立、网络照常运作，却时时被"说出口即随机成真"的愿望扰动。街头既有躲避怪潮的临时避难所，也有交易代价结晶的黑市。
- **愿望机制**：所有人说话都会被混乱至高神的"愿望哈希"截取，是否命中槽位取决于语义、情绪、代价承诺。成功的愿望可能引发代价回收，同时产出“混乱结晶”，失败什么也不会发生。
- **阵营格局**：乱神殿（骰子骑士、噪声祭司、代价银行）维持恐怖统治；秩序裁决局仅派出特遣队观测；民间有愿望猎人公会、乌鸦邮路等松散组织争夺生存空间。
- **关键资源**：混乱结晶是升级硬通货；"初始代码"为乱源真文的现代称呼，是破解哈希的底层脚本；代价银行掌握代价流向，可左右城市经济。
- **技术与风貌**：保留现代科技（监控、网络、便携终端），因此情报、直播、论坛推理都能成为剧情推力。

### 愿序等级（展示人与神差距）
- **无阶凡人**：无感知力，愿望完全随机，容易被代价收割。
- **拾愿（练气位）**：能感知愿力碎片，尝试控制一句话的"力度与指向"。
- **炼言（筑基位）**：凝聚"言核"，一天可主动引导一次愿望，需支付可测代价。
- **凝誓（结丹位）**：形成个人"愿望结界"，可限定他人愿望方向，同时承受反噬。
- **回响（元婴位）**：言出可影响小范围规则（如局部停时、禁声），开始接触神侍阶。
- **织律（化神位）**：可临时"织补规则"，对愿望哈希进行干涉，已能与低阶神侍对抗。
- **神侍/神使（合道/真仙位）**：神的代理与执法者，拥有半神位格；前期只出现影子。
- **至高神/主神**：仅存于神谕、记忆碎片中，保持"不可见但可感"的神秘高位。

### 势力与神的代理人
- **乱神殿**：外环的"骰子骑士"（执行强制愿望判决）、"噪声祭司"（散播愿望污染）、"代价银行"（回收代价结晶）。内圈吟唱者、神侍未现身，只投射神谕。
- **秩序道府**：符箓执事、镇狱使，来自秩序世界Y。任务是封锁混乱扩散；早期仅派一支特遣队入侵，暗中观察.
- **民间力量**：愿望猎人公会（半地下求生组织）、黑市情报链"乌鸦邮路"（鸦掌控的一条支线）、各城区武装帮派。
- **规则**：神不直接出面，所有命令通过神侍/神使传达，保持神秘与压迫感。

### 情报链与鸦设定
- **身份调整**：鸦曾是乱神殿的"代价记录官"（神侍序列末位），因偷走主神分裂时留下的记忆碎片被逐出神殿，降为凡阶伪装成情报商。
- **情报来源**：保留神殿档案口令+掌握黑市"乌鸦邮路"，同时暗中向秩序道府兜售情报，三方交换让他能触及机密。
- **动机**：以交易信息换取自身存在不被抹除，后期才暴露身为主神"后悔"碎片的真相。

### 升阶途径与风险（去中心化版本）
- **总则**：升级靠三条路径叠加——"积愿"（成功愿望堆叠混乱结晶）、"悟律"（发现新规律直接跨阶）、"献祭"（牺牲重要之物快速突破）。公会、神殿只是辅助记录者，真正的见证来自混乱世界本身：愿力会在瞬间化作浮现的"混乱符号"见证升阶，无需中心机构许可。
- **积愿机制**：每完成一次愿望（自用或代人），都会沉积一枚"混乱结晶"。结晶可存于个人、交易给黑市或代价银行，但至少保留70%才能稳定晋升，相当于常规、安全的升级路线。
- **悟律跳级**：谁能写下前所未有的愿望运算规律（例如"并行愿望互斥条款"），谁就能直接拉高阶位。悟律时刻由混乱之眼自动记载，公会最多只是替猎人出版规律手册。
- **献祭捷径**：献出至亲之物、记忆、甚至剩余寿命，可以强行换取阶位跃迁，但一旦反悔或代价不足，就会被代价银行标记为"失信体"并强制降级，形成悬念。
- **无阶→拾愿**：积累三枚基础混乱结晶即可感知愿力；或悟出第一条"愿望力度与指向"规律；或把最珍视的语言/名字献祭给混乱，以换取即刻感知。多数人通过生存任务自主完成，不需组织监督。
- **拾愿→炼言**：需要把混乱结晶压缩成"言核"。方式包括：①累积30次成功愿望，令结晶自发融合；②悟出影响广泛的第二规律（如"反向愿望的回声"）直接跳阶；③献祭记忆或视力换来言核，但会留下不可逆后遗症。
- **炼言→凝誓**：猎人需在现实中布置自己的"誓言坐标"（可以是老家院落、牺牲过的战场、或黑市祭坛），连续三次把愿望定向在同一区域，形成结界雏形；或者悟出覆盖范围规律（如"半径法则"）直接凝誓；又或者献祭血脉让结界绑定于亲人。
- **凝誓→回响**：要让誓言影响外界规则，可选择吞入完整混乱结晶群（积愿）、推演局部规则（悟律），或献祭声音/心跳作交换。不同路线对应不同副作用：积愿者稳定但慢，悟律者风险高，献祭者容易失控。
- **回响→织律**：需要接触原始规则底稿，但方法多样：有人靠积愿累积一百枚结晶买通代价银行的"规则碎片"，有人凭悟律解析神侍遗迹的符号，有人献祭未来十年的身份被秩序道府赦免。只要能把某条底层规则织入愿望哈希，就算成功。
- **织律→神侍/神使**：升华阶段依旧需要至高神的"代价契约"，替神执掌某段规则。过程伴随身份烙印与人性剥离，若违约即被抹除。神侍想保留自我就得通过代理人（如鸦）分拆记忆，这也是神侍影子稀少的缘由。
- **降级风险**：混乱结晶被抽空、悟律被证伪、献祭违约都会触发降级或失语。升阶越靠献祭与悟律，越需要后续验证，体现升级不仅是战力增长，还是持续赌注。

### 混秩等级映射
- **混乱世界 → 秩序世界**：无阶→凡体、拾愿→练气、炼言→筑基、凝誓→结丹、回响→元婴、织律→化神、神侍/神使→合道/真仙。
- **变量互认**：混乱愿力在秩序世界以"灵变量"形式出现；秩序符纹/道纹可视作混乱世界的"代价锚点"。两边体系对照，有助于角色在不同世界中快速判断战力与代价。

## 第一副本：混乱世界X（1-50章）

### 阶段一：底层求生与规则入门（1-10章）
- （第2章，林默日记视角）小镇怪潮+队友苏嘉因愿望失控死亡，林默写下"如果混乱也能被破解，我就不再恐惧"作为第一条悟律，展示**无阶凡人与拾愿**的巨大生存差距。
- （第5章，第三视角）救援任务中，林默让委托人说出"我愿意把痛全部拿走"并记录代价，三次成功愿望堆叠成第一枚混乱结晶（积愿路径）；旁观拾愿猎人惊呼"那孩子第一次许愿就敢赌这么狠？"
- （第10章，骰子骑士俯瞰视角）骰子骑士远望镇子，宣布"无序将被清算"，林默感到混乱符号在开眼，暗示升阶由世界见证而非公会许可；镇民跪地啜泣，反衬神侍威压。

### 阶段二：团队组建与等级曝露（11-20章）
- （第11章，林默视角）林默加入公会，师父级猎人一句"转身回去"让怪物倒退三步，顺带提醒徒弟："悟律满十条，混乱自会承认你是炼言。"被救的拾愿猎人直呼"这就是炼言的压迫感?"
- （第13章，苏黎视角）苏黎提前预警+楚寒符咒冻结，救下民众却引发"代价对等"（他城无辜者死亡），用苏黎视角强调规则残酷并展示她的共振悟律；幸存者哽咽问"我们活着，别人就得死吗？"
- （第15章，楚寒视角）公会猎人岳沐川献祭童年木马冲击凝誓，混乱符号瞬间熄灭，他被震回拾愿位，楚寒用冷静旁白记录其精神崩溃，警示献祭风险；围观猎人背脊发寒，互问"那我们还敢献吗？"
- （第18章，林默视角）噪声祭司播撒随机愿望，楚寒重伤。林默整合13条愿望笔记悟出"反向愿望回声"突破炼言，但代价是失声一昼夜（悟律路径成就+失败副作用）；噪声祭司阴笑："连声音都赌掉，还想跟神殿对抗？"

### 阶段三：愿望风暴与派系碰撞（21-30章）
- （第21章，阿九第一人称暴走视角）愿望风暴爆发，阿九咒语"愿你们忘了我"换来回响位力量（献祭记忆），她描述记忆剥离时的恐惧；楚寒听到她自述后沉默数秒，只回一句"别再这样赌了。"
- （第23章，林默视角）鸦首次交易，递上"风暴由神殿测试新哈希"的情报，提出交换林默笔记部分内容；路过的黑市客低声感叹"这鸦是真的知道神殿底牌"，强化悟律诱惑。
- （第26章，秩序裁决局档案视角）观察官洛衡之自称悟出"愿望互斥律"，冲击织律时话语被风暴撕碎，档案记录他从织律跌至拾愿的瞬间；裁决局同僚在密讯里写道："又一位以为自己能掌控混乱的傻子。"
- （第29章，竞技场全景视角）骰子骑士降临，林默喊出"愿你所判之骰有缺角"坑掉一次神谕，但立刻被碾压；观众席的民众齐声尖叫，骰子骑士冷笑："凡人，别以为缺角就会少死一个。"

### 阶段四：无常城外环试炼（31-40章）
- （第31章，小队多视角交错）公会派精英探索外环漏洞，队员约定各自升级路线：林默积愿+悟律，楚寒稳守积愿，苏黎唱诵悟律，阿九谨慎献祭，鸦半隐半现；鸦冷淡评价"同路不同道，别指望我救你们。"
- （第33章，苏黎目击视角）黑市织愿师席临岚在外环夜市献祭整条手臂，吟出"愿我血能织回规则"。苏黎听到骨骼崩裂声，夜市人潮倒吸凉气，有人喊"混乱不收赝品！"，埋下降阶伏笔。
- （第34章，代价银行巡视视角）队伍袭击代价收割队，画面中代价结晶被装入熔炉；林默用推理劫走整托结晶，获得凝誓所需的结晶群（积愿收益），被抓到的银行使者怒吼"代价不可盗！"
- （第36章，林默内心独白）他把结晶一次性献祭，试图跳阶凝誓，结果混乱符号倒转，结界崩塌，林默从炼言跌回拾愿；队友短暂沉默，鸦冷声点评："失败得漂亮，至少命还在。"
- （第38章，秩序裁决局监控视角）特遣队插手，楚寒身份暴露；织律裁决官一句"此街封口"冻结整区，迫使团队用混乱规则反制。围观群众被迫静止，只能用眼神传递恐惧。

### 阶段五：神侍影子的出现（41-50章）
- （第41章，阿九幻象视角）阿九体内"初始代码"触发，她看见主神分裂残影与污染源，幻象用第二人称对阿九说话，强化读者代入。
- （第44章，第三视角带旁白）鸦被旧同僚追杀，暴露神侍身份。团队救下他换得神侍遗迹钥匙，鸦顺势用残存悟律碎片示范织律干预；追杀者冷斥："叛侍无权再触规则！"
- （第48章，林默视角+苏黎歌声配合）林默在遗迹内重新凝聚言核：积愿重铸结晶，悟律补完"反噬倒流"公式，再献祭旧誓言中的恐惧记忆，终于稳住凝誓；苏黎歌声挡下神侍投影，投影冷声宣告："凡人，敢重写代价？"
- （第50章，神谕全知视角）遗迹出现至高神谕——"第零序列已启动"，暗示主神、至高神正在注视；团队被迫决定是否继续向神域迈进，为后续转折埋钩。

### 附：失败降阶与主角挫折
- **岳沐川（阶段二，第15章，楚寒侧记视角）**：他握着破旧木马低声祈求"愿我的童年可以换来结界"，献祭重量不足，混乱符号瞬间崩散，楚寒记录到他在血池里嚎哭，阶位跌回拾愿，提醒献祭路径风险极高。
- **洛衡之（阶段三，第26章，秩序裁决局档案视角）**：自创"愿望互斥律"妄图悟律跨越织律，却被风暴仪式判定伪命题；档案冷冷写道"其愿痕塌陷，位阶回落至拾愿"，示范悟律失败的严酷代价。
- **席临岚（阶段四，第33章，苏黎目击视角）**：她献祭左臂并喊出"愿我血能织回规则"，却因献祭对象与目标不符被代价银行反噬，阶位回跌至炼言，苏黎描述她的尖啸和崩塌，让读者第一时间体会恐惧。
- **林默（阶段四-五，第36-48章，林默笔记视角）**：急于凝誓时误判混乱符号，被打回拾愿，他在笔记写下"失败不是终点，只是新的公式"; 阶段五通过积愿重建结晶、悟律完善公式再献祭恐惧记忆，向读者展示主角失败也不放弃的韧性。
